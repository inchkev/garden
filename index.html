about.md (694B)

kevin.garden is my personal file gallery, fixated on the photos i've taken.

kevin.garden is also a mirror image of the underlying directory that hosts it:
html pages are generated by a program that crawls every file and parses
their actual finder positions within the directory.

view—and feel free to fork—the source code at github.com/inchkev/garden.
the repository paratactically hosts my personal kevin.garden, viewable
here, with the files and code that host and generate it. the .gardenignore
denylist prevents files such as favicon.png, CNAME, and node_modules/
from being shown.
