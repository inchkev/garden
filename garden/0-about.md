## key

dates

- "s" = sown from seed myself, "b" = bought
- "t" = transplanted
  - multiple indicates separate transplant dates
- "c" = cold stratification start
- "p" = picked a cutting from the world
- "rip" self explanatory
- if no year, assume 2025

tomatoes

- [i] indeterminate
- [s] semi-determinate
- [d] determinate
